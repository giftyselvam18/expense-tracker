# Expense Tracker Backend

A simple REST API for managing expenses using **Node.js**, **Express.js**, and **SQL Server**.

## Features
- Add Expense
- View Expenses
- Update Expense
- Delete Expense
- SQL Server Integration

## Tech Stack
- Node.js
- Express.js
- MongoDB

## Run the Project

```bash
npm install
npm start
```
